- ‚úÖ Purchase-based points (‚Ç±1 = 1 point base)
- ‚úÖ Tier multipliers (1x to 3x)
- ‚úÖ Birthday bonuses (future)
- ‚úÖ Referral rewards (future)

**Example:**
- Bronze customer spends ‚Ç±100 = 100 points
- Silver customer spends ‚Ç±100 = 150 points
- Gold customer spends ‚Ç±100 = 200 points
- Platinum customer spends ‚Ç±100 = 300 points

### Loyalty Transactions

**Transaction Types:**
```typescript
interface LoyaltyTransaction {
  id: string;
  customerId: string;
  tenantId: string;
  type: "earn" | "redeem" | "expire" | "adjustment";
  points: number;
  transactionId?: string;        // Link to sale
  reason: string;
  date: Date;
  expiryDate?: Date;             // Points expire in 1 year
}
```

### Rewards Catalog

**Available Rewards:**
```typescript
interface Reward {
  id: string;
  tenantId: string;
  name: string;
  description: string;
  pointsCost: number;            // Points needed
  cashValue: number;             // Actual value
  isActive: boolean;
  category: string;
  stock?: number;
}
```

**Sample Rewards:**
1. **‚Ç±50 Discount Voucher** - 500 points
2. **‚Ç±100 Discount Voucher** - 1,000 points
3. **Free Coke 1.5L** - 300 points
4. **‚Ç±200 Discount Voucher** - 2,000 points

### Redemption Process

1. Customer checks rewards catalog
2. Selects reward
3. System validates points balance
4. Deducts points
5. Generates voucher/applies discount
6. Records transaction
7. Updates customer points
